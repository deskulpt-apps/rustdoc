Register commands in a Deskulpt plugin.

This macro provides an automatic implementation of the Plugin::commands method. Each registered command must implement the PluginCommand trait.

ยงExample

use deskulpt_plugin::{register_commands, Plugin};

struct MyPlugin;

impl Plugin for MyPlugin {
    register_commands![/* List of commands to register */];
}
