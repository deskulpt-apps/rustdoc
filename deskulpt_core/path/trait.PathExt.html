Provided Methods§

Source

fn init_widgets_dir(&self) -> Result<()>

Initialize the widgets directory.

This will create the widgets directory if it does not exist. It must be called before calling the widgets_dir method.

Source

fn widgets_dir(&self) -> Result<&Path>

Get a reference to the widgets directory.

This will create the widgets directory if it does not exist, which can happen if one removes that directory during the application. This will error if the init_widgets_dir method has not been called.
