Used in BindingsBuilder::commands.

Collects function types into a [`Vec`], and all downstream types into a [`TypeCollection`](crate::TypeCollection) instance.

Specifying a type_map argument allows a custom [TypeCollection] to be used.

ยงExamples

use specta::*;

#[specta]
fn some_function(name: String, age: i32) -> bool {
    true
}

fn main() {
    let functions = function::collect_functions![some_function](&mut TypeCollection::default());
}
